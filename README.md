# Flappy-Bird

## Description

This project creates a clone of popular mobile game. **Flappy Bird**Â is a simple but addictive game where the player controls a bird that flies through a series of pipes. The player has to tap the screen or press the spacebar to make the bird flap its wings and avoid hitting the pipes. The game ends when the bird collides with a pipe or the ground. The goal is to fly as far as possible and get a high score.
